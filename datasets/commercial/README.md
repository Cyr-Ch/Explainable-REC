# Commercial Building Energy Profiles

This directory contains energy profiles for commercial building scenarios.

## Profiles

- **PV Profile**: Commercial solar generation pattern (larger system, peak during midday)
- **Load Profile**: Commercial consumption pattern (high during business hours, low at night)

## Usage

These profiles are used in `examples/scenarios/commercial_building.py`.

## Profile Characteristics

- Battery Capacity: 10.0 kWh (larger commercial battery)
- Battery Power: 5.0 kW
- Import Price: EUR 0.25/kWh
- Export Price: EUR 0.10/kWh
