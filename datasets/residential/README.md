# Residential Energy Profiles

This directory contains energy profiles for residential energy community scenarios.

## Profiles

- **PV Profile**: Typical residential solar generation pattern (peak during midday)
- **Load Profile**: Typical residential consumption pattern (higher in morning and evening)

## Usage

These profiles are used in `examples/scenarios/residential_energy_community.py`.

## Profile Characteristics

- Battery Capacity: 3.0 kWh (smaller residential battery)
- Battery Power: 1.5 kW
- Import Price: EUR 0.28/kWh
- Export Price: EUR 0.12/kWh
